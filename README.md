Σου επισυνάπτω τις σημειώσεις μου από το μάθημα "Βελτιστοποίηση". 

Σε παρακαλώ, μετάτρεψέ τες σε ένα οργανωμένο αρχείο Markdown (.md). 
Λειτούργησε ως έμπειρος βοηθός καθηγητή (TA): 
- Συμπλήρωσε το κείμενο με επεξηγήσεις, ορισμούς ή παραδείγματα όπου θεωρείς ότι οι σημειώσεις είναι ελλιπείς ή συνοπτικές (π.χ. πράγματα που ίσως δεν πρόλαβα να γράψω από τις διαφάνειες).
- Χρησιμοποίησε LaTeX για τους μαθηματικούς τύπους.
- Δομήσε το κείμενο με σαφείς επικεφαλίδες (##), λίστες και έντονα γράμματα (bold) για τις λέξεις-κλειδιά.

Σε παρακαλώ, τοποθέτησε ολόκληρο το τελικό κείμενο αυστηρά μέσα σε ένα Markdown code block (μπλοκ κώδικα). Έτσι θα μπορώ να το αντιγράψω με ένα κλικ και να το επικολλήσω στο δικό μου αρχείο .md

Αυτή πρέπει να είναι αυστηρά η μορφή του αρχείου:

---
title: "[Ένας σχετικός τίτλος για το σημερινό μάθημα]"
lesson: "[Όνομα μαθήματος, π.χ. Αναγνώριση Προτύπων και Στατιστική Μάθηση]"
date: "2026-03-11" 
tags: ["tag1", "tag2", "tag3", ...]
---
